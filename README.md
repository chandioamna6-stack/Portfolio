## üìÇ Getting Started

To run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/chandioamna6-stack/Portfolio.git
   ```

2. **Open the project**
   - Open `index.html` in your favorite browser.

3. **Configure EmailJS**
   - For contact functionality, sign up at [EmailJS](https://www.emailjs.com/) and add your EmailJS credentials to the corresponding JavaScript file as described in their [documentation](https://www.emailjs.com/docs/).
